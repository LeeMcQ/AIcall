About Sacred Wisdom

Sacred Wisdom is a Progressive Web App that runs a small language model entirely in your browser — no internet required after the first download, no servers, no data sharing.

The Research Model
This app is inspired by a fine-tuned SLM trained on the Bhagavad Gita, Quran, Bible, and Guru Granth Sahib. Read the article: Medium – Sandipan Das

Running that model on Android:

  1. Search Hugging Face for the model (search "sandipan religious text")
  2. Download a .gguf quantized file (Q4_K_M recommended for phones)
  3. Install PocketPal AI or SmolChat from the Play Store
  4. Load the .gguf file in the app and chat!

In-browser model (this app):

  1. Tap Load – the model downloads once (1–2.4 GB)
  2. It's cached in your browser for offline use
  3. Runs via WebGPU on your phone's GPU – no cloud needed

Requires Chrome 113+ on Android with WebGPU support. Samsung Internet and Firefox are not supported.

✦ All computation happens on your device ✦
