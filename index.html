Sacred Wisdom

Runs Google Gemma 3 or open-source models entirely in Chrome on your device. Zero data leaves your phone.

🌐 Google models (Gemma 3) — one-time setup: 1. Sign up free at huggingface.co
2. Accept Gemma license at Gemma3-1B-IT (and 4B if needed)
3. Get a free token at Settings → Tokens
4. Paste token → pick Gemma 3 1B → tap Load
⚡ WebLLM models — no token needed: Tap the WebLLM tab. Pick any model. Download starts immediately — no account required.
📲 Install as app: Chrome → ⋮ → Add to Home Screen. After first load the model is cached and works fully offline.

Which model for my phone?

  1. Any Android → Qwen 2.5 1.5B (WebLLM, no token, 1 GB)
  2. 4 GB+ RAM → Gemma 3 1B (Google, best quality, 0.7 GB)
  3. Flagship → Gemma 3 4B (Google, 2.4 GB)
